
# Express Starter kit

Express Starter for making beautiful API

## Tech Stack

**Server:** Express, Node

## Run Locally

Clone the project

```bash
  git clone git@github.com:Biblizar/express-starter.git
```

Go to the project directory

```bash
  cd express-starter
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
